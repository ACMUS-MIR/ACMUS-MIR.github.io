ACMUS-MIR DATASET

The ACMUS-MIR datas set is a growing collection of annotated music from the Andes region of Colombia. The data set was compiled with the goal of supporting the different MIR task addressed in our project: meter extraction, instrumental format recognition, and scale detection in music.

For each of these 3 tasks, a dedicated collection of music was compiled and annotated by musicologist in Colombia.

The ACMUS-MIR data set has been made publicly available for the wider research community as a Zenodo repository here. For a detailed description of the dataset, please refer to this publication.
